Welcome to DessieOwO3, the third design iteration of [dessieowo.com](https://www.dessieowo.com). 

DessieOwO3 completely changes how the site looks, and also cleans up a lot of bad code from when I was figuring things out. There are also some new features such as the new redirect feature, which redirects urls starting with /r/ to external sites (e.g. [www.dessieowo.com/r/github](https://www.dessieowo.com/r/github) redirects to [github.com/DessieOwO](https://github.com/DessieOwO)).

Though as of right now, DessieOwO3 is still under development. Please [contact us](https://www.dessieowo.com/about#contact) to let us know your opinions. If you discover an issue or have a suggestion, please [create an issue on github](https://github.com/Dessie-xyz/DessieOwO/issues/new).
